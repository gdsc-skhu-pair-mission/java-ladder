# 구현할 기능 목록 

## 입력
 - [ ] 사용자 이름 입력
   - [ ] 쉼표로 구분
   - [ ] 최대 5글자까지 가능
 - [ ] 사다리 높이 입력
   - [ ] 자연수


## 출력
  - [ ] 실행 결과
     - [ ] 첫번째 줄 : 사람 이름
     - [ ] 그 아래 줄 : 사다리 그리기

## 기능
- [ ] 사람의 이름을 받고 , 기준으로 나누기.
- [ ] 한 줄에 | 개수를 사람 이름 개수만큼 반환.
- [ ] 랜덤 값 기능 구현
- [ ] 랜덤에 충족하면 ----- 그리기.
  - [ ] 한줄의 |에 ----- 연결선이 동시에 나오지 않게
- [ ] 테스트 코드 작성
