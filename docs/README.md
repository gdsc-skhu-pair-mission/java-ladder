# 구현할 기능 목록 

## 입력
 - [x] 사용자 이름 입력
   - [x] 쉼표로 구분
   - [x] 최대 5글자까지 가능
 - [x] 사다리 높이 입력
   - [x] 자연수


## 출력
  - [x] 실행 결과
     - [x] 첫번째 줄 : 사람 이름
     - [x] 그 아래 줄 : 사다리 그리기

## 기능
- [x] 사람의 이름을 받고 , 기준으로 나누기.
- [x] 한 줄에 | 개수를 사람 이름 개수만큼 반환.
- [x] 랜덤 값 기능 구현
- [x] 랜덤에 충족하면 ----- 그리기.
  - [x] 한줄의 |에 ----- 연결선이 동시에 나오지 않게
- [x] 테스트 코드 작성
